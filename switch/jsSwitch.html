Switch Statement

Async

Switch statement is used to execute one code from multiple expressions. It is just like else if statement that we have learned in previous page. But it is convenient than if..else..if because it can be used with numbers, characters etc.

Switch cases use strict comparison (===). The values must be of the same type to match. A strict comparison can only be true if the operands are of the same type.

  • Syntax:
    switch(expression) {
     case value1:
      code to be executed;
     break;
     case value2:
      code to be executed;
     break;
    ......

     default:
      code to be executed if above values are not matched;
    }
