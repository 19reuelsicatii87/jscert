Without Constructor

A JavaScript class is not an object. It is a template for JavaScript objects. Classes serve as blueprints for creaing objects with predefined properties and methods.

  • Syntax:
    class className {
     prop1;
     prop2;
     method_1() {
       //code here
     }
    }

With Constructor

A JavaScript class is not an object. It is a template for JavaScript objects. Classes serve as blueprints for creaing objects with predefined properties and methods.

  • Syntax:

      constructor(param1, param2) {
      this.param1 = param1;
      this.param1 = param1;
     }
     method_1() {
       //code here
