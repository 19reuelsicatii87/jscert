Exporting / Importing Module

Static Import

JavaScript modules allow you to break up your code into separate files.This makes it easier to maintain a code-base.Modules are imported from external files with the import statement.

**Static Import**:
The import happens statically within the `script type="module` tag. This means that the module is loaded as soon as the browser encounters the tag during HTML parsing, regardless of whether it is actually needed or not. This approach is suitable for modules that are essential for the initial page load and execution.

JavaScript modules allow you to break up your code into separate files.This makes it easier to maintain a code-base.Modules are imported from external files with the import statement.

**Dynamic Import**:
The import is triggered by a function call (`myFunctionDynamicImport()`), which happens when the button is clicked. This allows for lazy loading of the module, which can be useful for optimizing page load times by loading modules only when needed.
