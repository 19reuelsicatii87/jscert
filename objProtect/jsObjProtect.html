Lesson: Object Protection

Object protection typically refers to techniques or mechanisms used to prevent unintentional or unauthorized modification of object properties. This can include preventing the addition, deletion, or modification of properties within an object

Object.preventExtensions() and Object.isExtensible()

Object.seal() and Object.isSeal()

Object.seal(): Array

The Object.seal() method seals an object, preventing new properties from being added and existing properties from being deleted. However, existing properties can still be modified.
